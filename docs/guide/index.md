# Example UI component

Example ui component documentation

## Quick start

1. Install the package:

```bash [pnpm]
pnpm add @monorepo/ui
```

2. Import  component:

```ts
import { uiExample } from '@monorepo/ui'
```

3. Use the `ui-example` component:

```html
<ui-example text="This is example component" />
```
